Contract Monthly Claim System (CMCS)
Project Overview:
A comprehensive ASP.NET Core MVC web application that automates monthly claim submissions and approvals for Independent Contractor Lecturers. Features role-based access control and automated workflows.
Module: PROG6212 - Programming 2B

Key Features:
Lecturer
Automated claim submission with real-time calculations
Document upload with validation
Claim tracking with status updates

Coordinator
Automated claim review dashboard
One-click approval/rejection system
Status workflow automation

Manager
Final approval system
Automated filtering of coordinator-approved claims
Complete audit trail

HR Administrator (Enhanced Feature)
User management system
Role-based user creation
Report generation

Technology Stack:
Backend: ASP.NET Core MVC, Entity Framework, SQL Server
Frontend: Bootstrap 5, jQuery, Razor Pages
Security: ASP.NET Identity, Role-based Authorization
