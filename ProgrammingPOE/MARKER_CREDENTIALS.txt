=== SYSTEM FEATURES ===

1. LECTURER VIEW:
   - Automatic hourly rate population from user profile
   - Auto-calculation of total amount (Hours × Rate)
   - File upload support for documents
   - Claim tracking with status

2. COORDINATOR VIEW:
   - Review pending claims
   - Approve or reject claims
   - Two-step approval workflow

3. MANAGER VIEW:
   - Final approval of coordinator-approved claims
   - Cannot see pending or rejected claims

4. HR VIEW:
   - User management (create/edit users)
   - Report generation
   - System oversight

=== TECHNICAL DETAILS ===
- Database: SQL Server Express
- Authentication: ASP.NET Identity
- Framework: ASP.NET Core
- Auto-calculation: JavaScript + Server-side validation
- File Upload: Maximum 5MB, various formats supported
